# DMVPN_IKEv2


mGRE - Multipoint Generic Routing Encapsulation
	 NHRP - Next Hop Routing Protocol
	 NSH  - Next Hop Server
	 Overlay IP   - Tunnel Public Address
	 Underlay IP  - Tunnel Private Address
	 EIGRP Route - show ip route eigrp




Show Command:

	 #sh ip nhrp
	 #sh crypto ipsec sa
   #sh crypto ikev2 sa
	 #sh ip eigrp int br
	 #sh ip eigrp nei
